A journal that actually knows you.

You talk, it listens. It notices when you drift, nudges you on the days you need it, and helps you go deeper when something is stuck. Encrypted on your laptop. No accounts, no cloud, no passwords to lose.

Install (3 steps)

1

Clone the repo

Copy this and paste it into your terminal. You need git and python 3.10+.

git clone https://github.com/parthreddy123/agni.git
cd agni
2

Install

This makes the agni command available everywhere. Pulls in cryptography as the only dependency.

pip install -e .
3

Initialize

Generates a local encryption key. No passphrase to remember. Done.

agni init

Optional but recommended — install the /agni slash command for Claude Code so you can use Agni as a copilot:

mkdir -p ~/.claude/commands && cp .claude/commands/agni.md ~/.claude/commands/

What you get — six modes

Daily
A 5-minute structured journal. Whatever's on your mind, three things you're grateful for, three intentions, an affirmation. Or a free-form brain dump.
Therapy
Sit down and talk. Synthesize-first: Claude asks what patterns YOU have noticed before sharing its own read. Your brain does the work.
Challenge
An adversarial session. Claude picks something you wrote this week and pushes back. You defend it or refine it. Uncomfortable by design — counterweight to AI agreement.
Exercises
19 single-session practices across 5 categories. Add "cold" for a no-warm-up version where Claude stays silent until you finish.
Profile
A 36-question enneagram test. Takes 10 minutes. Once set, it becomes the lens for every other mode — sharper prompts, sharper observations.
Dashboard
Streak, 90-day dot grid, and a read of where you are. Not stats for stats' sake.
Vault
Export everything as plain markdown so you can browse by date in Obsidian, OneNote, or any notes app.

Smart nudges

Agni knows what to ask for based on local time and what you've already done today. No global schedule, no cron jobs — it just reads your state when you talk to it.

morning_due
It's morning local time and the morning journal hasn't happened. Opens with morning mode.
evening_due
It's evening, morning was done, evening hasn't been. Closes out the day.
therapy_weekend
Sat/Sun and you haven't sat down for therapy in 6+ days. Opens proactively.
exercise_stale
10+ days since any standalone exercise. Recommends one based on your enneagram type.
low_consistency
2 or fewer entries in the last 7 days. Names the drift gently.
daily_due
It's evening and nothing has been logged. Suggests a quick recovery — half is better than nothing.

How storage works

Entries live in ~/.agni/ as encrypted .jrnl files. The encryption key is auto-generated on first run and stored locally — no passphrase to forget. This protects against cloud sync exposure and casual filesystem browsing. For full protection on a stolen laptop, enable disk encryption on your machine.

Run agni vault sync any time to export everything as plain markdown, organized by date, ready to open in Obsidian or any notes app.

Quick reference

agni daily              structured 5-minute journal
agni profile test       enneagram (10 min, do once)
agni exercise list      browse all 19 exercises
agni exercise run X     run a specific one
agni streak             90-day dot grid
agni read 2026-04-15    view a specific day
agni vault sync         export to ~/Documents/agni-vault
